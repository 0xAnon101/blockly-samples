This sample shows the minumum code needed to define a new renderer that sets connection shapes at runtime based on a connection's type checks:

  1. Define a renderer that extends the base renderer: TypedConnectionShapeRenderer
  2. Register the renderer by name: 'typed_connection_shapes'
  3. Define a constants provider that extends the base constants provider: TypedConnectionShapeProvider. It contains all of the constants needed to size a block during rendering.
  4. Call the superclass constructor to set default property values.
  5. Override makeConstants_ to tell the custom renderer to use the custom constants provider.
  6. Define the makeSquared and makeRounded functions to return rectangular and rounded shapes respectively.
  7. Override the init function on the TypedConnectionShapeProvider:
    • Call the superclass init function to set up default properties.
    • Create and save the new connection shapes in this.SQUARED and this.ROUNDED
  8. Override the shapeFor function to inspect the connection's type checks array and return the correct connection shape:
    • Return a rounded tab for inputs and outputs that accept numbers and strings.
    • Return a squared tab for other inputs and outputs.
    • Return the normal notch for previous and next connections.
