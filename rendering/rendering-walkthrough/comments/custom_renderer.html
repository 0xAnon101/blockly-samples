The minimum code needed to define a new renderer:

  1. Define a renderer that extends the base renderer: CustomRenderer
  2. Register the renderer by name: 'custom_renderer'

The resulting renderer looks like the base renderer.

Note that this renderer looks unfinished when compared to Geras. Some areas have too much padding, and others have too little.

You can extend any of the built-in renderers by changing line 7 to refer to the desired renderer.
